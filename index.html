ScheduleMsg

Productivity · Automation

ScheduleMsg helps users schedule messages and automatically deliver them at the perfect time. Whether it's a birthday greeting, reminder, follow-up message, or important notification, ScheduleMsg ensures every message reaches the right person at the right moment.

Supported Platforms

  • SMS
  • Facebook Messenger
  • Instagram

Features

  • Auto-send support

03 — Web Projects

Web Projects

Web platforms designed for discovery and ease of use.

LogoHub

Web Platform · Directory

LogoHub is a visual company directory where users can explore businesses through logos. Each company logo appears inside a colored category box. Clicking a logo opens company information and provides direct access to the company's website.

How It Works

  1. Browse logo grid
  2. Click company logo
  3. View company details
  4. Visit official website

Color Categories

  • Blue — Technology
  • Pink — Entertainment
  • Teal — Healthcare
  • Green — Food & Travel

Features

  • Logo-based navigation
  • Category color system
  • Company information popup
  • Fast website access
  • Company submission system
  • Responsive design
